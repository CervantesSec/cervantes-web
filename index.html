About Cervantes

Cervantes is an opensource collaborative platform for pentesters or red teams who want to save time to manage their projects, clients, vulnerabilities and reports in one place.

  • Cervantes is OpenSource and available on GitHub and has an easy setup
  • A docker container with a pre-built version of Cervantes is available.

Take Cervantes for a spin and review the demo of Cervantes and login with sample credentials.

Frequently Asked Questions

  • Whats is Cervantes' relationship with OWASP?

    Cervantes is an OWASP Project

  • Why build an application like this?

    Cervantes exists because there are not many applications like Cervantes that assist you in managing your project, clients, reports generation in only one place

  • Deploy options?

    Cervantes can be deplyoyed in less than a minute via Docker or it can be deployed via the source code.

  • Cervantes support customized templates?

    Yes. You can upload your customized templates via Report Templates in dotx format

  • Is 2FA supported on Cervantes?

    Yes, you can set the 2FA with you favourite 2FA app.
